Possibilities of the VOB system

  • One cable

    One cable replaces the function of many cables currently used on the market. Data transmission and power supply to devices in one cable

  • Configuration

    Real-time device configuration depending on user needs

  • Standard power

    Standard power supply up to 3600W at 230V and the ability to adapt the cable to individual needs

  • Speed

    Information transfer speed of 2.4 Gb/s which will increase as technology develops

  • Long-range

    No drop in data transfer speed when using the VOB system over long distances between the transmitter and receiver

  • Resistant

    Any damage to the cable or part of the installation will disconnect the power supply in the section for which it was responsible, and the system will indicate the location and type of damage

  • Universal

    Possibility to replace all existing connectors, such as: HDMI, USB/C, Rj45. Through one universal connector or using adapters. It does not force you to change your current devices

  • Centralized

    One control panel connecting and communicating all systems and devices available on the market today

  • Secure

    The shielded cable blocks electromagnetic smog

  • Flexible

    Very easy installation, modification and expansion of the system

  • Cyber secure

    The structure of the cable itself protects the system against data loss or theft, and its universality allows it to be used in any industry and broadly understood services

What exactly is VOB

The core of the VOB system is innovative cable, or more precisely its cross-section. Placed in the central part optical fiber enables the transfer of information. Further copper braids enable the supply of electricity. And the outer braid shields andprotects entire cable. Simple design with endless possibilities.

All you need to create any installation is passive distributors and universal terminal boxes. Any compact device such as a camera, lamp, smoke detector, or light switch can be installed in the boxes; or install any type of connector such as HDMI, USB, antenna, power socket, etc.

All the most important elements to create a VOB system are available on the market. Thanks to this, the implementation of VOB technology will not be so expensive and time-consuming. The element managing all devices in the installation is the VOB control panel. One control panel performs the same function as all the others combined in classic systems.

Data transmission is based on the existing and generally available GPON technology in the ITU-T G.984.x standard at a speed of 2.4 Gb/s. The cable enables electricity to be supplied to the receiver with a power of up to 3.6kW, which is also standard in Europe.

The innovation is a new type of plug that connects the fiber optic part of the cable with devices through a "contact" without the need for "welding" (permanent connection). Plus a plugin connects at any angle to the receiver's socket.
